Adds Button to your Editor to insert HR Tag <hr> (Horizntal line or Rule ) anywhere, and also includes easy to use 18 different horizontal line styles to choose from with just one click.

== Description ==
This plugin adds a simple Button in Editor to insert HR Rule (Horizontal Line) and also includes awesome looking HR styles to pick from, by just choosing any style you like.


**Multisite Supported:** Tested to work fine in Multisite network.

== Installation ==
1. Use Wordpress' plugin interface to find and install the plugin.
3. Under Appearance -> HR Styles Menu . Click the checkbox on style which you like for your HR lines.
2. Start editing the post or page where you want to add HR line by clicking new HR button .
4. Save changes.
